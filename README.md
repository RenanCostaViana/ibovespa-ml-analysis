# 📈 Ibovespa ML Analysis — Market Data Analytics

Análise do mercado financeiro brasileiro utilizando Machine Learning  
Projeto de análise exploratória e experimentação preditiva com dados históricos do Ibovespa

---

## 🧠 Visão Geral

Este projeto tem como objetivo analisar dados históricos do mercado acionário brasileiro (Ibovespa),
aplicando técnicas de **análise de dados**, **engenharia de atributos** e **Machine Learning** para
explorar padrões, relações estatísticas e limitações preditivas em séries temporais financeiras.

O foco do projeto é **analítico e educacional**, buscando compreender o comportamento dos dados
e avaliar o uso de modelos de ML em um contexto notoriamente complexo e ruidoso, como o mercado financeiro.

> ⚠️ Este projeto **não constitui recomendação de investimento** e não tem como objetivo prever
ou sugerir operações de compra ou venda.

---

## 🎯 Objetivos do Projeto

- Explorar dados históricos do Ibovespa e compreender sua estrutura
- Realizar análise exploratória (EDA) de preços, retornos e variações
- Criar atributos derivados relevantes para séries temporais
- Aplicar modelos de Machine Learning de forma experimental
- Avaliar o desempenho e as limitações dos modelos em dados financeiros
- Documentar o processo de forma clara, crítica e responsável

---

## 📊 Etapas da Análise

1. **Aquisição e entendimento dos dados**
   - Série histórica do índice
   - Organização temporal e tratamento inicial

2. **Análise Exploratória (EDA)**
   - Comportamento de preços e retornos
   - Distribuição, volatilidade e padrões temporais
   - Visualizações para interpretação do mercado

3. **Engenharia de Atributos**
   - Retornos percentuais
   - Lags e janelas móveis
   - Indicadores derivados para apoio à análise

4. **Modelagem com Machine Learning**
   - Teste de modelos supervisionados como ferramenta analítica
   - Avaliação de desempenho
   - Comparação entre abordagens

5. **Interpretação dos Resultados**
   - Análise crítica dos resultados obtidos
   - Discussão sobre ruído, não‑estacionariedade e limites preditivos

## 🧠 Principais Aprendizados

- Dados financeiros são altamente ruidosos e não estacionários
- O uso de ML em mercado financeiro exige cautela e senso crítico
- Métricas tradicionais nem sempre refletem valor preditivo real
- A engenharia de atributos tem impacto maior do que a complexidade do modelo
- Análise responsável é mais importante do que promessas de previsão

---

## ⚠️ Limitações

- O modelo não considera fatores externos (macro, notícias, eventos)
- Resultados são sensíveis ao período analisado
- O projeto não foi desenvolvido para uso operacional ou decisões reais
- Machine Learning é tratado como **ferramenta de análise**, não como solução definitiva
